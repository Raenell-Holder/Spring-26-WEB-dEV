Taking a screenshot

If you are working on a Windows machine like I am, the simplest way to take a screenshot is to use the print screen built-in button on the keyboard. To most, this button would look like prt sc. So simply navigate to your keyboard, find this button, click on it and then select the portion of your screen you would like to capture and "screen capture" it!
